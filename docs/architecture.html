🔒 Authentication Architecture

1

User Authentication (JWT)

End users login with email/password. Backend issues HS256 JWT with 24hr expiry. Stored in localStorage, sent as Authorization: Bearer header on every request. JwtTokenFilter validates on each request.

2

Service-to-Service (API Key)

Portfolio API calls Agent Farm using AGENT_FARM_API_KEY (default: admin-secret). Health check (3s timeout) determines availability before each AI request.

3

MCP Consumer Auth + Subscription

Agent Farm calls MCP Gateway with consumer API key. Gateway checks consumers table, validates subscriptions (canRead/canExecute per MCP server). Rate limiting: 200 req/min.

4

Admin Auth

Admin Nexus uses ADMIN_API_KEY stored in browser localStorage. Same key authenticates against Gateway and Agent Farm. Grants access to management routes.

5

MCP Server → Backend Auth

Portfolio MCP server authenticates to Portfolio API using a service account JWT. Logs in on startup, caches token, uses it for all API calls.

6

Ownership Validation

verifyPortfolioOwnership() on all AI endpoints. Extracts userId from SecurityContext, verifies against portfolio.userId. ADMIN role bypasses.

7

External APIs

Anthropic Claude: x-api-key header. Zerodha Kite: OAuth 2.0 with API key + secret. Brave Search: API key.

💰 Tax Calculation Engine

FIFO Lot Matching Algorithm
Input: FY "2025-26" → April 2025 to March 2026

SELL TRADES IN FY:
  RELIANCE  2025-06-10  10 shares @ ₹2,900
  TCS       2025-11-20   5 shares @ ₹3,800

BUY TRADES (FIFO QUEUE):
  RELIANCE  2025-01-15  10 shares @ ₹2,650.50
  TCS       2025-03-20   5 shares @ ₹3,450.00

FIFO MATCHING:
  ┌─────────────────────────────────────────────────────────┐
  │ RELIANCE: BUY Jan 15 → SELL Jun 10                      │
  │   146 days held (< 365) → STCG                         │
  │   Gain: (₹2,900 - ₹2,650.50) × 10 = ₹2,495            │
  │                                                          │
  │ TCS: BUY Mar 20 → SELL Nov 20                           │
  │   245 days held (< 365) → STCG                         │
  │   Gain: (₹3,800 - ₹3,450) × 5 = ₹1,750                │
  └─────────────────────────────────────────────────────────┘

TAX COMPUTATION:
  LTCG (>365d):  ₹0      Tax: ₹0     (12.5% above ₹1.25L)
  STCG (≤365d): ₹4,245   Tax: ₹849   (20% flat)

INCOME AGGREGATION:
  Dividends:   ₹5,000   TDS: ₹500
  FD Interest: ₹12,000  TDS: ₹1,200
  ───────────────────────────────
  Total: ₹21,245    TDS: ₹1,700

🛡 Resilience Patterns

AI Fallback

Agent Farm (MCP tools, richer context) → Direct Claude API (inline schema). 3s health check timeout determines path.

Circuit Breaker

MCP Gateway client pool: 5 failures → 60s open → half-open retry. Stale session eviction on ECONNREFUSED.

Task Queue

BullMQ: 5 concurrent workers, 3 retries with exponential backoff (2s base), 120s agent timeout, max 10 tool steps.

Data Import

Kite API sync (real-time) → CSV file upload → Paste CSV text (mobile). Three fallback levels for data input.

Health Monitoring

30s health check loop for all registered MCP servers. Status tracked in database. Auto-reconnect on recovery.

Tradebook Dedup

order_id uniqueness check prevents duplicate imports. Same CSV uploaded twice → 0 created, N skipped.

💡 Design Principles

Apps own data, Agent Farm owns intelligence

Portfolio API manages domain data; Agent Farm orchestrates all AI. Clean separation.

User context via tool parameters

userId passed explicitly to MCP tools. Language-agnostic, no JWT forwarding complexity.

Defense in depth

7 authentication layers from JWT to ownership validation. Each layer independent.

Graceful degradation

Agent Farm down → Claude fallback. Kite down → CSV import. Always functional.

Full transparency

Every AI analysis persisted with model, prompt, raw response for complete audit trail.

FIFO tax compliance

Tax engine uses legally required First-In-First-Out lot matching per Indian regulations.
